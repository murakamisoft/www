VSCodeを日本語化する方法

VSCodeを使い始めたばかりだと、メニューや設定がすべて英語で戸惑うことも多いはずです。公式の「日本語化パック」を導入して、快適な開発環境を手に入れましょう。

1. 日本語化の概要

VSCodeは世界中で利用されており、Microsoft公式から「Japanese Language Pack」が無料で提供されています。これをインストールするだけで、メニューやダイアログが日本語に切り替わります。

項目 内容
対象 Windows / macOS / Linux
費用 完全無料
所要時間 約1分

2. 日本語化の手順

2-1. 拡張機能ビューを開く

VSCode左側のサイドバーにある「Extensions（四角いアイコン）」をクリックするか、ショートカットキー Ctrl + Shift + X を押します。

2-2. 検索とインストール

検索バーに「Japanese」と入力し、Microsoftから提供されている「Japanese Language Pack for Visual Studio Code」を探して「Install」ボタンを押します。

3. 設定の確認と反映

インストールが完了すると、右下に「Restart」というボタンが表示されます。これを押すとVSCodeが再起動し、自動的に日本語表示に切り替わります。 


# もし反映されない場合はコマンドパレット(Ctrl+Shift+P)から
# "Configure Display Language" と入力して選択してください

4. うまく日本語にならない場合

再起動しても英語のままの場合は、以下の設定ファイル locale.json が正しく更新されているか確認してください。 


{
  "locale": "ja"
}

5. まとめ

  • 拡張機能から「Japanese Language Pack」を検索する。
  • インストール後にVSCodeを再起動するだけ。
  • 表示が変わらない場合は「Configure Display Language」を確認する。
