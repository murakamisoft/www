VSCodeでJava開発環境を構築する方法

VSCodeは軽量でありながら、拡張機能を導入することで強力なJava開発環境に変貌します。本記事では、JDKの導入からVSCodeでのプロジェクト作成まで、Spring Boot開発にも対応したステップを解説します。

1. 概要

VSCodeでJavaを扱うためには、まずJDKをインストールし、VSCodeにJava用の拡張機能パックを導入する必要があります。

項目 内容
環境 Java 21 (推奨)
必須拡張機能 Extension Pack for Java
ビルドツール Maven または Gradle

2. 手順

2-1. JDKのインストール

公式サイトまたはAdoptium(Temurin)からJDK 21をダウンロードし、パスを通してください。 


# インストール確認コマンド
java -version

2-2. VSCode拡張機能の導入

VSCodeの拡張機能タブで「Extension Pack for Java」を検索し、インストールします。これにより、コード補完やデバッグ機能が自動的に有効になります。

3. サンプルコード

環境構築後に作成する最小限のコントローラー例です。


@RestController
public class HelloController {

  @GetMapping("/")
  public String hello() {
    return "Hello, VSCode Java!";
  }

}

4. よくあるエラー

JAVA_HOMEが設定されていない

VSCodeがJavaを見つけられない場合、環境変数「JAVA_HOME」を確認してください。Windowsの場合はシステム環境変数の編集画面から設定可能です。 


# パスが通っているか確認
echo %JAVA_HOME%

5. まとめ

  • JDK 21をインストールし、環境変数を通す。
  • 「Extension Pack for Java」をVSCodeへ導入する。
  • Maven/Gradleを使ってプロジェクトを管理する。
