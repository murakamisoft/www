VB6で作成したミニゲームのソースコードを紹介

はじめに

このサイトでは、Visual Basic 6.0を使った簡単なミニゲームのソースコードを紹介します。
2Dベースで、BitBltという高速画像転送を使用したゲームを紹介します
現在はバリバリ3Dで魅力的なゲームであふれています。しかし昔にはやった素朴な2Dのゲームも、まだまだ捨てたものじゃありません。
どうせなら自分で作ってみませんか？
作る過程を楽しみ、できあがったらフリーソフトとして公開し、出来がよければシェアウェアとして販売してしまいましょう。

ある程度VBをいじった事はあるけれども、ゲームを作ったことがなく、これから作ろうかな？という初心者の方を対象としています。
すべてのソースコードを紹介していますので、当サイトで何らかのヒントが得られれば幸いです。

必要なものは、開発環境として、Visual Basic 6.0と、ゲームで使用するドット絵を描くソフトです。
VB6はフリーではありませんが、ドット絵を描くソフトはフリーで優れたものがたくさんあります。ネットで探してみてください。

プログラムのソースコードをすべて公開していますので、自由に改変していただいてかまいません。

姉妹サイトのJavaでコンソールアプリを作ろうも宜しくお願いします。
