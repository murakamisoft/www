Docker Composeが動かない主な原因

原因 内容
yaml構文エラー インデントミス
依存関係 DB起動前にアプリ起動
ポート競合 8080等が使用中
環境変数 .env不足

調査手順

docker compose ps
docker compose logs
docker compose config

サンプル設定

services:
  app:
    build: .
    depends_on:
      - db

  db:
    image: postgres:16

よくあるエラー

yaml: mapping values are not allowed here

インデントミスを確認してください。

port is already allocated

ポート競合です。

まとめ

  • docker compose configで構文確認
  • logsで原因調査
  • ポート競合確認
  • 依存関係を見直す
