WindowsでSpring Boot開発環境を構築する手順（Java21 + VS Code + Gradle + PostgreSQL）

Windows 11にSpring Boot開発環境を構築する手順を紹介します。 Java 21、VS Code、Gradle、PostgreSQLを使用し、Spring Bootアプリケーションを起動できるところまで解説します。

対象読者

  • Spring Bootをこれから学習したい人
  • EclipseではなくVS Codeを利用したい人
  • Java 21環境を構築したい人
  • PostgreSQLを利用したWebアプリを開発したい人

今回構築する環境

ソフトウェア バージョン
Windows 11
Java 21 (LTS)
VS Code 最新版
Gradle Spring Boot標準
PostgreSQL 17系
Spring Boot 3.x

1. Java 21をインストールする

まずJava 21をインストールします。 Spring Boot 3系ではJava 17以上が必要です。 今回は長期サポート版（LTS）のJava 21を利用します。

インストール後、コマンドプロンプトで以下を実行します。

java -version

以下のように表示されれば成功です。

openjdk version "21"

2. VS Codeをインストールする

Visual Studio Codeをインストールします。

インストール後、以下の拡張機能を追加します。

  • Extension Pack for Java
  • Spring Boot Extension Pack
  • Gradle for Java

これらを導入することでSpring Boot開発に必要な機能が利用できます。

3. PostgreSQLをインストールする

PostgreSQLをインストールします。

インストール時に以下を設定します。

  • postgresユーザーのパスワード
  • ポート番号（通常は5432）

接続確認を行います。

psql -U postgres

ログインできれば成功です。

4. Spring Bootプロジェクトを作成する

Spring Initializrを利用してプロジェクトを作成します。

以下の設定を選択します。

項目 設定値
Project Gradle
Language Java
Spring Boot 最新版
Packaging Jar
Java 21

Dependenciesには以下を追加します。

  • Spring Web
  • Spring Data JPA
  • PostgreSQL Driver
  • Lombok

5. VS Codeでプロジェクトを開く

ダウンロードしたプロジェクトを展開します。

sample-project

VS Codeでフォルダを開きます。

初回起動時はGradle依存ライブラリのダウンロードが実行されます。

6. PostgreSQL接続設定を行う

application.propertiesを編集します。


	spring.datasource.url=jdbc:postgresql://localhost:5432/sampledb
	spring.datasource.username=postgres
	spring.datasource.password=password

	spring.jpa.hibernate.ddl-auto=update
	spring.jpa.show-sql=true

事前にデータベースを作成しておきます。


	CREATE DATABASE sampledb;

7. Spring Bootを起動する

VS Codeのターミナルから以下を実行します。

gradlew bootRun

LinuxやMacの場合は以下です。

./gradlew bootRun

以下のメッセージが表示されれば起動成功です。


	Started SampleApplication

8. 動作確認

ブラウザから以下へアクセスします。

http://localhost:8080

エラーが表示されなければSpring Bootは正常に起動しています。

よくあるエラー

javaコマンドが見つからない


	'java' は内部コマンドまたは外部コマンド...

PATH設定を確認してください。

PostgreSQLに接続できない


	Connection refused

PostgreSQLサービスが起動しているか確認してください。

ポート8080が使用中


	Port 8080 was already in use

他のSpring Bootアプリが起動していないか確認してください。

まとめ

WindowsでSpring Boot開発環境を構築する手順を紹介しました。

  1. Java 21をインストール
  2. VS Codeをインストール
  3. PostgreSQLをインストール
  4. Spring Initializrでプロジェクト作成
  5. Gradleで起動

ここまで完了すれば、Spring BootによるWebアプリ開発を始められます。
